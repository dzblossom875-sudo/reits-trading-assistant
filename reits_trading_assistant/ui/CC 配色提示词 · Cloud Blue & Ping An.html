请使用以下 CSS 变量配色方案（Cloud Blue）绘制图表，配色粘贴进 HTML <style> 中： :root { --bg:#f2f6fb; --surface:#ffffff; --surface-2:#e4edf8; --border:#cddaee; --s1:#1a4a80; --s2:#d44000; --s3:#00806a; --s4:#7040a0; --s5:#c08800; --s6:#006080; --s1-fill:rgba(26,74,128,0.08); --s2-fill:rgba(212,64,0,0.08); --s3-fill:rgba(0,128,106,0.08); --bull:#00806a; --bull-bg:#e0f4f0; --bear:#d44000; --bear-bg:#faeee6; --text:#0a1428; --text-muted:#5a7aa0; --text-dim:#9ab0cc; --grid:rgba(26,74,128,0.05); --grid-major:rgba(26,74,128,0.10); --axis:#cddaee; } 图表要求： - 背景色 var(--bg)，卡片白底 var(--surface)，边框 var(--border) - 折线/柱颜色依次用 var(--s1) var(--s2) var(--s3)，面积填充用对应 --s*-fill - 网格线用 var(--grid-major)，轴刻度文字用 var(--text-muted) - 正值/上涨标签：background var(--bull-bg)，color var(--bull) - 负值/下跌标签：background var(--bear-bg)，color var(--bear) - Chart.js 示例写法：borderColor:'#1a4a80', backgroundColor:'rgba(26,74,128,0.08)' [在此替换你的具体图表需求，例如： "双折线图，x轴月份2025.01-2026.03，s1=沪深300，s2=标普500"]
